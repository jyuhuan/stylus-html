This is a tutorial of how to do academic writing with Stylushtml. Stylushtml is the HTML/JavaScript implementation of Stylus. It serves as the assembly for all other (future) Stylus implementations. Even so, Stylushtml itself is simple and human-friendly enough to be used directly.
Stylushtml provides a set of convenient tags for declaratively specifying the various aspects of your text. The section describes these tags and their semantics.
In general, a document (termed paper) will have the following structure: ... ...

If a section is positioned inside another section, the latter will be treated as a subsection of the former.
